### 问题 B:

## Experiment 3

### 问题 A
使用ITIMER_REAL型定时器实现一个gettimeofday()，将它设置为每秒产生一个信号，
并计算已经经过的秒数。

### 问题 B
使用以上实现的 gettimeofday()实现一个精确到微秒级的“壁钟”。

### 问题 C
实现以上一个父进程和两个子进程并发递归计算不同项数的 fibonacci 序列的程序，
分析每个进程三种类型定时器测出的时间关系。
